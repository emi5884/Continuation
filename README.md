## 🎨  コンセプトとデザイン
- コンセプト ... 継続は進化なり。
- カラー ... ヒヨコ&ニワトリ(進化を体現する動物)のカラーである黄、赤、白を基調色としています。
- スタンプカード風デザイン ... 過去の記録が一目でわかるようラジオ体操のスタンプカードのようなデザインにしました。画像がスタンプ代わりです。
